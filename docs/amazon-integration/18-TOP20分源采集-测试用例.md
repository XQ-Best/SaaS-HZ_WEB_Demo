### 自动化断言（并入 audit 脚本）

```python
def audit_top20_coverage(rows, limit=20):
    top = sorted(rows, key=lambda r: parse_money(r["revenue_30d"]), reverse=True)[:limit]
    n = len(top) or 1
    return {
        "revenue_pct": sum(1 for r in top if r["revenue_30d"]) / n,
        "orders_pct": sum(1 for r in top if int(r["orders_30d"] or 0) > 0) / n,
        "ad_spend_pct": sum(1 for r in top if r["ad_spend_30d"]) / n,
    }
    # FAIL if revenue_pct < 0.9 after successful br_csv sync
```
